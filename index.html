Prerequisites

  • Java 8 or newer
    • see how to install Java 17 for help on windows or use java -version in terminal
  • Minecraft Java Edition 1.21
    • see Mincraft 1.21

Server Subdomain

use server address: Friend.mc.gg

Recommended

  • Fabric Loader 0.15.11 or newer for Minecraft 1.21 Client
    • see Fabric Installer
  • Allocate 4GB Ram to Minecraft Installation
    • Change -Xmx2G in JVM ARGUMENTS to -Xmx4G and click Save
  • Add Mods to %appdata%\.minecraft\mods
    • or wherever your .minecraft/mods folder is located (different for Mac and Linux)

Optionally

  • Add a Resource Pack to %appdata%\.minecraft\resourcepacks
    • The Server resource pack is not enforced, so feel free to disable it in server info
    • Direct download redirect link: here
  • Add Sodium and Indium or any other client side performance enhancing mods to your mods folder
    • see Indium
    • Note: While Sodium does not require Indium API, the Continuity mod requires Indium when Sodium is installed
